# Tree-Diagram

## Features

- **Multi-language Support**: English, Vietnamese, Chinese, France, Russian
- **Theme Support**: Light/Dark mode
- **Drag & Drop**: Move nodes around the canvas
- **Zoom & Pan**: Zoom in/out and pan the view
- **Auto Layout**: Automatically arrange nodes in a tree structure
- **Node Operations**: Add, edit, delete nodes
- **Edge Operations**: Add, edit, delete edges
- **Export/Import**: Export/import tree data as local formatted file extenstion (.dt)

## Bản Tiếng Việt

## Tính năng

- **Hỗ trợ đa ngôn ngữ**: Tiếng Anh, tiếng Việt, tiếng Trung, tiếng Pháp, tiếng Nga
- **Hỗ trợ chủ đề**: Chế độ sáng/tối
- **Kéo và thả**: Di chuyển các nút trên khung vẽ
- **Phóng to/thu nhỏ**: Phóng to/thu nhỏ và di chuyển khung nhìn
- **Tự động sắp xếp**: Tự động sắp xếp các nút theo cấu trúc cây
- **Thao tác với nút**: Thêm, chỉnh sửa, xóa nút
- **Thao tác với cạnh**: Thêm, chỉnh sửa, xóa cạnh
- **Xuất/Nhập**: Xuất/nhập dữ liệu cây dưới dạng tệp có định dạng cục bộ (.dt)
